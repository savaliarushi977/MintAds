## Glossary

| Term | Definition |
|------|------------|
| **Experience ID** | A Headout catalog identifier for a single bookable experience (e.g., "colosseum-skip-the-line-arena-floor-tour"). The primary input to the pipeline. Any valid Headout ID is accepted. |
| **facts.json** | The normalized catalog data object for one experience. Single source of truth for all downstream claims. Produced by the Content Ingestion stage from Headout's public APIs. |
| **script.json** | The creative blueprint for one ad variant. Produced by the Script Engine (Claude). Contains both `video_script` (scene-by-scene visual directions) and `audio_script` (VO segments synced to scenes). |
| **Angle** | The strategic argument an ad is built to land (e.g., A3 = "Don't Waste Your Trip / Skip-the-Line"). From Gokul's playbook. User selects via dropdown. |
| **Hook** | The scroll-stopping opening mechanic (first 1–2 seconds). Same angle can have many hooks — the cheapest variable to test. User selects via dropdown. |
| **Beat** | One of four structural units in every ad: HOOK (0–5s), BODY (5–17s), PAYOFF (17–23s), CTA/END CARD (23–27s). Fixed at 3 content beats + 1 end card. |
| **Segment / Scene** | One video generation unit. Maps 1:1 to a beat. Each scene = one Seedance API call = one raw video clip. |
| **Variant** | One complete ad creative: a unique combination of experience × angle × hook × format. One pipeline run = one variant. |
| **Ad ID** | The naming convention string that uniquely identifies a variant: `HDO_META_{POI}_{Angle}_{Hook}_{Format}_{Lang}_{Version}`. |
| **End Card** | The branded closing frame (3–5s). NOT generated by Seedance — rendered as a Remotion React component. Contains price, rating, CTA, logo from facts.json. |
| **Master Clock** | The VO audio duration. In assembly, video clips are trimmed/padded to match audio timing. Audio always wins timing conflicts. |
| **Claim Source** | A mapping from an on-screen factual assertion (price, rating, duration) to the specific facts.json field it came from. Enforced by the script validator. Zero orphan claims allowed. |
| **Run** | A single end-to-end pipeline execution for one variant: ingestion → script → video + audio → assembly → export. |
| **Cost Ledger** | Per-run accumulation of all external API costs (Claude, Seedance, ElevenLabs). Target: < $10 per variant. |

## 0. Summary

Headout sells 10,000+ experiences. Fewer than 50 have dedicated ad creative because production costs $300–3,000 per asset and takes weeks per cycle. MintAds automates this: an experience ID goes in, campaign-ready UGC-style video ads come out.

The pipeline has 7 stages: **User Input → Content Ingestion → Script Engine → Video Engine → Audio Engine → Assembly → Export**. Video and Audio run in parallel after Script. Assembly waits for both. The whole chain takes ~2–3 minutes per variant and costs ~$3–6.

**Three components**: the Express backend (orchestrator — pipeline coordination, external API calls, DB, cost tracking), the React frontend (UI — form, progress, playback, downloads), and the Remotion renderer (assembly — video + audio + overlays + end card → final MP4).

**Key numbers**: 3 Seedance calls + 1 ElevenLabs call + 1 Claude call per variant. ~20 seconds of video output. 3 aspect ratios per variant (9:16, 1:1, 16:9). Target: 5 experiences × 2 angles × 2 hooks = 20 variants for the demo, under $120 total.

**The unfair advantage**: every on-screen claim traces to Headout's catalog via the claim validator. Off-the-shelf tools generate pretty videos with hallucinated facts. This engine generates UGC-style ads that are *true*.

## 2. Technical Decisions

| Decision | Choice | Rationale |
|---|---|---|
| Language | TypeScript end-to-end | Remotion is React/TS native; one runtime avoids Python↔Node handoff |
| Frontend | Plain React + Vite | Fastest to scaffold; no SSR needed. Separate from Express backend. |
| Backend | Express.js | Lightweight, familiar, no framework overhead for a 42-hour hack |
| Database | PostgreSQL (Replit native) | Queryable, structured, supports JSONB for flexible schema areas |
| ORM | Raw `pg` client with parameterized queries | No Prisma migration overhead; ~7 tables don't justify an ORM |
| File storage | Local filesystem (`/data/runs/{ad_id}/`) | Video files are 5–50MB; filesystem is fast, inspectable, servable via Express static. Paths stored in PostgreSQL. |
| Video generation | Higgsfield SDK (`@higgsfield/client` v2) with Seedance 2.0 I2V | Direct subscription on ai-dev@headout; SDK handles polling; per-scene generation (Option B) |
| Audio generation | ElevenLabs SDK (`@elevenlabs/elevenlabs-js`) | Official Node SDK; `textToSpeech.convert()` for file output; `convertWithTimestamps()` for word-level sync (Phase 2) |
| Script generation | Claude API (`claude-sonnet-4-6`) | Strong instruction-following for structured JSON + claim tracing enforcement |
| Assembly | Remotion (`@remotion/renderer`) | React-based video composition; programmatic rendering via `renderMedia()`; bundle once, render many |
| Assembly fallback | ffmpeg CLI | If Remotion fails on Replit (memory/Chrome), fall back to `ffmpeg -i clip1.mp4 -i clip2.mp4 -filter_complex concat` |
| Video gen per scene | Option B — individual clips per scene | Granular control, per-scene retry, reuse body/payoff across hook variants |
| Seedance audio | `generate_audio: false` | Avoid conflicting ambient audio across stitched segments; ElevenLabs VO is the single audio track |
| Synthetic humans | In scope | UGC creative direction requires on-camera AI creator presence |
| Ad structure | Fixed 3 content segments + 1 end card = ~20s | Predictable orchestration, consistent Remotion composition, predictable cost |
| Aspect ratio strategy | Generate master 9:16 video clips; Remotion handles 1:1 and 16:9 via cropping/reframing | 3 Seedance calls per variant, not 9; saves ~$6 per variant |
| VO approach | Single continuous VO (Phase 1); per-segment VO (Phase 2 if sync quality demands it) | One ElevenLabs call; no audio seam artifacts; script engine writes beat-aware VO |
| Image selection | Claude reads photo `alt`/`keyword` metadata from facts.json; assigns `photo_reference_index` per scene | No vision model needed; Headout API provides descriptive metadata per image |
| LLM call count | One Claude call per variant (script gen); images NOT sent to Claude | Input is ~1,600 tokens, output is ~1,000 tokens = ~$0.02. No benefit to multi-call or vision. |
| Cost tracking | Per-API-call logging in `stage_logs` table; running total on `runs` table | Automatic; every service wrapper calls `trackCost()` after success |
| Auth | None (hackathon) | Internal demo tool; no user accounts needed |
| Progress tracking | Frontend polls `GET /api/status/:ad_id` every 2 seconds | Simple, reliable, no WebSocket complexity for a hack |
| Naming convention | `HDO_META_{POI}_{Angle}_{Hook}_UGC_{Lang}_{Version}` | Attributable in analytics; matches Gokul's playbook convention |

- `additional_details`: optional string, max 500 chars

#### Script Validator (runs automatically before proceeding)

```typescript
interface ValidationResult {
  passed: boolean;
  violations: string[];
}

function validateScript(script: ScriptJson, facts: FactsJson): ValidationResult {
  const violations: string[] = [];

  // CHECK 1: Structural validity
  if (!script.video_script?.scenes?.length) {
    violations.push('video_script.scenes is empty or missing');
  } else {
    if (script.video_script.scenes.length !== 3) {
      violations.push(`Expected 3 scenes, got ${script.video_script.scenes.length}`);
    }
    for (const scene of script.video_script.scenes) {
      if (!scene.scene_id || !scene.beat || !scene.duration_sec || !scene.visual_direction) {
        violations.push(`Scene ${scene.scene_id}: missing required fields`);
      }
      if (scene.photo_reference_index !== null &&
          (scene.photo_reference_index < 0 || scene.photo_reference_index >= facts.photos.length)) {
        violations.push(`Scene ${scene.scene_id}: photo_reference_index ${scene.photo_reference_index} out of range (0-${facts.photos.length - 1})`);
      }
    }
    const totalDuration = script.video_script.scenes.reduce((s, sc) => s + sc.duration_sec, 0);
    if (totalDuration < 15 || totalDuration > 23) {
      violations.push(`Total scene duration ${totalDuration}s outside 15-23s range`);
    }
  }

  if (!script.audio_script?.vo_segments?.length) {
    violations.push('audio_script.vo_segments is empty or missing');
  } else {
    for (const seg of script.audio_script.vo_segments) {
      if (!seg.scene_id || !seg.vo_text || !seg.target_duration_sec) {
        violations.push(`VO segment ${seg.scene_id}: missing required fields`);
      }
      const matchingScene = script.video_script?.scenes?.find(s => s.scene_id === seg.scene_id);
      if (!matchingScene) {
        violations.push(`VO segment ${seg.scene_id}: no matching video scene`);
      } else if (matchingScene.duration_sec !== seg.target_duration_sec) {
        violations.push(`VO segment ${seg.scene_id}: duration ${seg.target_duration_sec}s ≠ scene duration ${matchingScene.duration_sec}s`);
      }
    }
  }

  if (!script.end_card?.price_display || !script.end_card?.rating_display || !script.end_card?.cta_text) {
    violations.push('end_card missing required fields (price_display, rating_display, cta_text)');
  }

  // CHECK 2: Claim completeness
  if (!script.claim_sources || Object.keys(script.claim_sources).length === 0) {
    violations.push('claim_sources is empty — every factual claim must be traced');
  } else {
    for (const [claim, sourcePath] of Object.entries(script.claim_sources)) {
      if (sourcePath.startsWith('facts.')) {
        const resolved = resolveJsonPath(facts, sourcePath.replace('facts.', ''));
        if (resolved === undefined) {
          violations.push(`Claim "${claim}" references non-existent field "${sourcePath}"`);
        }
      }
      // Allow non-facts sources like "config" or "known queue time" — these are acceptable
    }
  }

  // CHECK 3: Orphan claim detection
  const allText = [
    ...script.audio_script.vo_segments.map(s => s.vo_text),
    ...script.video_script.scenes.map(s => s.text_overlay).filter(Boolean),
    script.end_card?.price_display,
    script.end_card?.rating_display,
    script.end_card?.review_count_display,
    script.end_card?.cancellation_text
  ].filter(Boolean).join(' ');

  // Extract factual assertions: currency amounts, ratings, counts, durations
  const factPatterns = [
    /[$€£¥]\d+[\d,.]*/g,                    // currency: $68, €45
    /\d+[\d,.]*[$€£¥]/g,                    // currency suffix: 68€
    /\d+\.\d\s*[★☆]/g,                      // ratings: 4.6★
    /\d+\.\d\/\d/g,                          // ratings: 4.6/5
    /[\d,]+\+?\s*reviews?/gi,               // review counts: 28,540+ reviews
    /\d+[-–]\d+\s*(hours?|minutes?|min)/gi, // durations: 1.5-2 hours
  ];

  const claimKeys = Object.keys(script.claim_sources).map(k => k.toLowerCase());
  for (const pattern of factPatterns) {
    const matches = allText.match(pattern) || [];
    for (const match of matches) {
      const normalized = match.toLowerCase().trim();
      const isClaimed = claimKeys.some(k => k.includes(normalized) || normalized.includes(k));
      if (!isClaimed) {
        violations.push(`Orphan claim detected: "${match}" found in script text but not in claim_sources`);
      }
    }
  }

  return { passed: violations.length === 0, violations };
}

function resolveJsonPath(obj: any, path: string): any {
  return path.split('.').reduce((o, k) => o?.[k], obj);
}
```

#### Auto-Retry Logic

```typescript
async function generateAndValidateScript(
  facts: FactsJson,
  userInput: UserInput,
  angleDesc: string,
  hookDesc: string,
  runId: number,
  adId: string
): Promise<ScriptJson> {
  const MAX_ATTEMPTS = 2;

  for (let attempt = 1; attempt <= MAX_ATTEMPTS; attempt++) {
    // Update stage log
    await updateStageLog(runId, adId, 'script_gen', 'in_progress', attempt);

    // Generate script
    let prompt = buildScriptPrompt(facts, userInput, angleDesc, hookDesc);
    if (attempt > 1 && previousViolations) {
      prompt += `\n\nYOUR PREVIOUS SCRIPT HAD THESE VIOLATIONS:\n${previousViolations.join('\n')}\n\nFix all violations. Every claim must trace to facts.json.`;
    }

    const response = await callClaude(prompt, SCRIPT_ENGINE_SYSTEM_PROMPT);
    const script = JSON.parse(response);
    const cost = calculateClaudeCost(response);
    await trackCost(runId, adId, 'script_gen', 'claude-sonnet-4-6', cost, { attempt, tokens: response.usage });

    // Validate
    const validation = validateScript(script, facts);
    await updateStageLog(runId, adId, 'script_validation', validation.passed ? 'completed' : 'failed', {
      attempt, passed: validation.passed, violations: validation.violations
    });

    if (validation.passed) {
      // Build claim report
      const claimReport = buildClaimReport(script, facts);
      await db.query('UPDATE runs SET script = $1, claim_report = $2 WHERE id = $3',
        [JSON.stringify(script), JSON.stringify(claimReport), runId]);
      return script;
    }

    previousViolations = validation.violations;
  }

  throw new Error(`Script validation failed after ${MAX_ATTEMPTS} attempts: ${previousViolations.join('; ')}`);
}
```

### 3.3 Video Engine (Backend → Higgsfield SDK → Video Clips)

**Trigger**: Script validated successfully. Runs in PARALLEL with Audio Engine.

**What happens**: 3 separate Higgsfield API calls, one per scene, all fired in parallel via `Promise.all`. Each call takes a product photo URL + scene visual direction and returns a video clip.

```typescript
import { higgsfield, config } from '@higgsfield/client/v2';

config({ credentials: process.env.HIGGSFIELD_CREDENTIALS }); // KEY_ID:KEY_SECRET

interface VideoGenInput {
  scene_id: number;
  beat: string;
  prompt: string;
  image_url: string;
  duration: number;
  aspect_ratio: string;
}

async function generateVideoClip(input: VideoGenInput): Promise<VideoClipResult> {
  const jobSet = await higgsfield.subscribe('seedance-v2.0-i2v', {
    input: {
      prompt: input.prompt,
      images_list: [input.image_url],
      aspect_ratio: input.aspect_ratio, // "9:16" for master
      duration: input.duration,
      quality: input.beat === 'hook' ? 'high' : 'basic', // hero hook gets high quality
      generate_audio: false, // VO handled by ElevenLabs
    },
    withPolling: true,
  });

  if (!jobSet.isCompleted || !jobSet.jobs[0]?.results?.raw?.url) {
    throw new Error(`Seedance generation failed for scene ${input.scene_id}: ${jobSet.jobs[0]?.error || 'unknown'}`);
  }

  const videoUrl = jobSet.jobs[0].results.raw.url;

  // Download to local filesystem
  const localPath = `/data/runs/${adId}/clips/clip_${String(input.scene_id).padStart(3, '0')}.mp4`;
  await downloadFile(videoUrl, localPath);

  return {
    scene_id: input.scene_id,
    beat: input.beat,
    file_path: localPath,
    remote_url: videoUrl,
    duration_sec: input.duration, // actual duration may differ slightly
  };
}

// Orchestrator calls all 3 scenes in parallel
async function generateAllVideoClips(
  script: ScriptJson,
  facts: FactsJson,
  userInput: UserInput,
  runId: number,
  adId: string
): Promise<VideoClipResult[]> {

  const sceneInputs: VideoGenInput[] = script.video_script.scenes
    .filter(s => s.beat !== 'cta') // end card is Remotion-generated, not Seedance
    .map(scene => ({
      scene_id: scene.scene_id,
      beat: scene.beat,
      prompt: scene.visual_direction,
      image_url: facts.photos[scene.photo_reference_index]?.url || facts.photos[0].url,
      duration: scene.duration_sec,
      aspect_ratio: '9:16', // master aspect ratio — Remotion handles 1:1, 16:9
    }));

  // Fire all in parallel
  const results = await Promise.allSettled(
    sceneInputs.map(async (input) => {
      await updateStageLog(runId, adId, `video_gen_scene_${input.scene_id}`, 'in_progress');
      try {
        const clip = await generateVideoClip(input);

        // Sanity check: file exists, non-zero bytes
        const stats = await fs.stat(clip.file_path);
        if (stats.size === 0) throw new Error('Generated clip is 0 bytes');

        const cost = input.duration * (input.beat === 'hook' ? 0.30 : 0.15); // high vs basic
        await trackCost(runId, adId, `video_gen_scene_${input.scene_id}`, 'higgsfield',
          cost, { scene_id: input.scene_id, duration: input.duration, quality: input.beat === 'hook' ? 'high' : 'basic' });

        await updateStageLog(runId, adId, `video_gen_scene_${input.scene_id}`, 'completed');
        return clip;
      } catch (error) {
        // Retry once
        try {
          const clip = await generateVideoClip(input);
          await trackCost(runId, adId, `video_gen_scene_${input.scene_id}`, 'higgsfield',
            input.duration * 0.15, { scene_id: input.scene_id, retry: true });
          await updateStageLog(runId, adId, `video_gen_scene_${input.scene_id}`, 'completed', { retried: true });
          return clip;
        } catch (retryError) {
          await updateStageLog(runId, adId, `video_gen_scene_${input.scene_id}`, 'failed', { error: retryError.message });
          throw retryError;
        }
      }
    })
  );

  // Collect successful clips; allow partial (assembly can work with fewer)
  const clips = results
    .filter((r): r is PromiseFulfilledResult<VideoClipResult> => r.status === 'fulfilled')
    .map(r => r.value);

  if (clips.length === 0) {
    throw new Error('All video generation calls failed');
  }

  return clips.sort((a, b) => a.scene_id - b.scene_id);
}
```

**Timing**: ~60–120 seconds per clip. Since all 3 run in parallel, total wall time ≈ slowest clip.

**Cost per variant (video)**: 3 clips × ~6s average × $0.15–0.30/sec = **$2.70–5.40**

### 3.4 Audio Engine (Backend → ElevenLabs SDK → VO Audio)

**Trigger**: Script validated successfully. Runs in PARALLEL with Video Engine.

**What happens**: One ElevenLabs call generates the complete VO as a single continuous audio file.

```typescript
import { ElevenLabsClient } from '@elevenlabs/elevenlabs-js';

const elevenlabs = new ElevenLabsClient({
  apiKey: process.env.ELEVENLABS_API_KEY
});

// Hardcoded for hackathon — Gokul picks during manual testing tonight
const VOICE_ID = process.env.ELEVENLABS_VOICE_ID || 'JBFqnCBsd6RMkjVDRZzb';
const MODEL_ID = 'eleven_multilingual_v2';

async function generateVoiceover(
  script: ScriptJson,
  runId: number,
  adId: string
): Promise<VoiceoverResult> {
  await updateStageLog(runId, adId, 'audio_gen', 'in_progress');

  // Concatenate all VO segments into one flowing text
  // The script engine wrote them as beat-aware segments with natural pacing cues
  const fullVoText = script.audio_script.vo_segments
    .map(seg => seg.vo_text)
    .join(' ');

  const audio = await elevenlabs.textToSpeech.convert(VOICE_ID, {
    text: fullVoText,
    model_id: MODEL_ID,
    voice_settings: {
      stability: 0.35,         // lower = more expressive (UGC energy)
      similarity_boost: 0.75,
      style: 0.5,
      use_speaker_boost: true,
    },
  });

  // Write audio stream to file
  const localPath = `/data/runs/${adId}/vo_audio.mp3`;
  const writeStream = createWriteStream(localPath);
  for await (const chunk of audio) {
    writeStream.write(chunk);
  }
  writeStream.end();
  await new Promise(resolve => writeStream.on('finish', resolve));

  // Sanity check: file exists, non-zero bytes
  const stats = await fs.stat(localPath);
  if (stats.size === 0) throw new Error('VO audio file is 0 bytes');

  // Get actual duration via ffprobe
  const actualDuration = await getAudioDuration(localPath);
  const targetDuration = script.audio_script.total_duration_target_sec;

  if (Math.abs(actualDuration - targetDuration) > 5) {
    console.warn(`VO duration ${actualDuration}s deviates >5s from target ${targetDuration}s`);
  }

  // Cost: ~$0.03 per 1K characters
  const charCount = fullVoText.length;
  const cost = (charCount / 1000) * 0.03;
  await trackCost(runId, adId, 'audio_gen', 'elevenlabs', cost,
    { characters: charCount, model: MODEL_ID, voice_id: VOICE_ID, actual_duration: actualDuration });

  await updateStageLog(runId, adId, 'audio_gen', 'completed',
    { duration_sec: actualDuration, characters: charCount });

  return {
    file_path: localPath,
    duration_sec: actualDuration,
    characters: charCount,
  };
}
```

**Cost per variant (audio)**: ~350 chars × $0.03/1K = **~$0.01**

```tsx
import { AbsoluteFill, Sequence, Audio, OffthreadVideo, Img, useCurrentFrame, interpolate } from 'remotion';

interface AdProps {
  clips: Array<{ src: string; durationSec: number; beat: string }>;
  voAudioSrc: string;

    cancellationText: string | null;

export const AdComposition: React.FC<AdProps> = ({
  clips, voAudioSrc, textOverlays, endCard, fps
}) => {
  let currentFrame = 0;

  return (
    <AbsoluteFill style={{ backgroundColor: 'black' }}>
      {/* Video clips sequenced by scene order */}
      {clips.map((clip, i) => {
        const from = currentFrame;
        const durationFrames = Math.round(clip.durationSec * fps);
        currentFrame += durationFrames;
        return (
          <Sequence from={from} durationInFrames={durationFrames} key={`clip-${i}`}>
            <OffthreadVideo
              src={clip.src}

      <Audio src={voAudioSrc} />

          from={Math.round(overlay.startSec * fps)}
          durationInFrames={Math.round(overlay.durationSec * fps)}
          key={`text-${i}`}
```

```typescript
async function initRemotionBundle() {

// Aspect ratio configs
const ASPECT_CONFIGS = {
  '9:16': { width: 1080, height: 1920 },
  '1:1':  { width: 1080, height: 1080 },
  '16:9': { width: 1920, height: 1080 },

async function assembleAd(
  clips: VideoClipResult[],
  voAudio: VoiceoverResult,
  script: ScriptJson,
  userInput: UserInput,
  runId: number,
  adId: string
): Promise<AssemblyResult> {
  await updateStageLog(runId, adId, 'assembly', 'in_progress');

  // Build input props for Remotion
  const inputProps: AdProps = {
    clips: clips.map(c => ({
      src: c.file_path,
      durationSec: c.duration_sec,
      beat: c.beat,
    })),
    voAudioSrc: voAudio.file_path,
    textOverlays: script.video_script.scenes.map(scene => {
      const cumulativeStart = script.video_script.scenes
        .filter(s => s.scene_id < scene.scene_id)
        .reduce((sum, s) => sum + s.duration_sec, 0);
      return {
        text: scene.text_overlay || '',
        startSec: cumulativeStart,
        durationSec: scene.duration_sec,
      };
    }),
    endCard: {
      priceDisplay: script.end_card.price_display,
      ratingDisplay: script.end_card.rating_display,
      reviewCountDisplay: script.end_card.review_count_display,
      ctaText: script.end_card.cta_text,
      brandLogo: script.end_card.brand_logo,
      cancellationText: script.end_card.cancellation_text,
    },
    fps: 30,
  };

  // Total duration = sum of scene durations + 4s end card
  const totalDurationSec = script.video_script.total_duration_sec + 4;
  const totalFrames = Math.round(totalDurationSec * 30);

  // Determine which formats to render
  const formats = userInput.video_format === 'all'
    ? ['9:16', '1:1', '16:9']
    : [userInput.video_format];

  const outputFiles: OutputFile[] = [];

  for (const format of formats) {
    const { width, height } = ASPECT_CONFIGS[format];
    const outputPath = `/data/runs/${adId}/output/${format.replace(':', 'x')}.mp4`;
    await fs.mkdir(path.dirname(outputPath), { recursive: true });

    const composition = await selectComposition({
      serveUrl: bundleLocation,
      id: 'AdComposition',
      inputProps,
    });

    await renderMedia({
      composition: {
        ...composition,
        width,
        height,
        durationInFrames: totalFrames,
        fps: 30,
      },
      serveUrl: bundleLocation,
      codec: 'h264',
      outputLocation: outputPath,
      inputProps,
    });

    // Sanity check
    const stats = await fs.stat(outputPath);
    if (stats.size === 0) throw new Error(`Rendered ${format} is 0 bytes`);

    outputFiles.push({
      format,
      file_path: outputPath,
      file_size: stats.size,
      duration_sec: totalDurationSec,
    });
  }

  await trackCost(runId, adId, 'assembly', 'remotion', 0, { formats_rendered: formats.length });
  await updateStageLog(runId, adId, 'assembly', 'completed', { formats: formats });

  return { files: outputFiles };
}
```

## 7. Orchestrator — Full Pipeline Code

```typescript
// orchestrator.ts — the core pipeline runner

async function runPipeline(userInput: UserInput): Promise<string> {
  // 1. Generate Ad ID
  const adId = generateAdId(userInput);
  
  // 2. Create run record
  const run = await db.query(
    `INSERT INTO runs (ad_id, experience_id, persona, journey_type, brand, angle_id, hook_id, video_format, additional_details, status, current_stage)
     VALUES ($1,$2,$3,$4,$5,$6,$7,$8,$9,'pending','content_ingestion')
     RETURNING id`,
    [adId, userInput.experience_id, userInput.persona, userInput.journey_type, userInput.brand, userInput.angle, userInput.hook, userInput.video_format, userInput.additional_details]
  );
  const runId = run.rows[0].id;

  try {
    // 3. Content Ingestion
    await updateRunStatus(runId, 'ingesting', 'content_ingestion');
    const facts = await fetchExperienceFacts(userInput.experience_id, runId, adId);

    // 4. Script Generation + Validation
    await updateRunStatus(runId, 'scripting', 'script_gen');
    const angleDesc = await getAngleDescription(userInput.angle);
    const hookDesc = await getHookDescription(userInput.hook);
    const script = await generateAndValidateScript(facts, userInput, angleDesc, hookDesc, runId, adId);

    // 5. Video + Audio in PARALLEL
    await updateRunStatus(runId, 'generating', 'video_gen + audio_gen');
    const [videoClips, voAudio] = await Promise.all([
      generateAllVideoClips(script, facts, userInput, runId, adId),
      generateVoiceover(script, runId, adId),
    ]);

    // 6. Assembly
    await updateRunStatus(runId, 'assembling', 'assembly');
    const assembly = await assembleAd(videoClips, voAudio, script, userInput, runId, adId);

    // 7. Export & Finalize
    await updateRunStatus(runId, 'exporting', 'export');
    await exportAndFinalize(assembly, script, facts, script.claim_report, runId, adId);

    await updateRunStatus(runId, 'completed');
    return adId;

  } catch (error) {
    await db.query(
      `UPDATE runs SET status = 'failed', error_message = $1, completed_at = NOW() WHERE id = $2`,
      [error.message, runId]
    );
    throw error;
  }
}

function generateAdId(input: UserInput): string {
  const poi = input.experience_id
    .split('-').slice(0, 2).join('')
    .replace(/[^a-zA-Z]/g, '')
    .substring(0, 15);
  const version = `v${String(Date.now()).slice(-4)}`;
  return `HDO_META_${poi}_${input.angle}_${input.hook}_UGC_EN_${version}`;
}

async function updateRunStatus(runId: number, status: string, stage?: string) {
  await db.query(
    `UPDATE runs SET status = $1, current_stage = COALESCE($2, current_stage) WHERE id = $3`,
    [status, stage, runId]
  );
}
```

## 8. Error Handling Matrix

| Scenario | Detection | Run Status | Stage | User Sees |
|---|---|---|---|---|
| Invalid Experience ID | Headout API returns 404 | failed | content_ingestion | "Experience not found. Check the ID and try again." |
| Headout API timeout | No response after 10s + 1 retry | failed | content_ingestion | "Could not reach Headout catalog. Try again." |
| Missing critical facts (no price, no photos) | Validation after fetch | failed | content_ingestion | "Missing required data: price, photos. Cannot generate." |
| Claude returns invalid JSON | JSON.parse throws | retried once | script_gen | If retry fails: "Script generation failed. Try different inputs." |
| Script fails claim validation (attempt 1) | Validator returns violations | retried | script_validation | Invisible — auto-retry with violation feedback |
| Script fails claim validation (attempt 2) | Validator returns violations | failed | script_validation | "Script could not pass claim verification: {violations}" |
| Seedance returns error for one scene | API error response | partial | video_gen_scene_N | If 2/3 clips succeed: assembly continues with fewer clips. If 0/3: failed. |
| Seedance returns 0-byte clip | File size check | retried once | video_gen_scene_N | If retry fails, scene skipped; assembly uses remaining clips |
| All 3 Seedance calls fail | All promises rejected | failed | video_gen | "Video generation failed for all scenes. Check Higgsfield status." |
| ElevenLabs timeout | No response after 30s | retried once | audio_gen | If retry fails: "Audio generation failed." |
| VO duration off by >5s | Duration check after generation | warning | audio_gen | Logged as warning; assembly proceeds (audio is master clock) |
| Remotion render fails (memory) | renderMedia throws | attempted ffmpeg fallback | assembly | If ffmpeg also fails: "Video assembly failed." |
| Remotion bundle missing | bundleLocation is null | failed | assembly | "Remotion not initialized. Server restart required." |
| Final video is 0 bytes | File size check | failed | assembly | "Rendered video is empty. Assembly error." |
| Cost exceeds $10 per variant | Running total check after each stage | warning | any | Logged but not blocked — budget is a target, not a hard stop |
| Demo: live generation fails | Any stage failure | failed | any | Switch to pre-cached output: "Here's what it produced earlier." |
